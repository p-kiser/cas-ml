```
01_intro/                   Python, NumPy, Pandas, data visualisation
02_supervised_learning/     Regression, classification, preprocessing
03_unsupervised_learning/   Clustering, dimensionality reduction
04_deep_learning/           Neural networks, Keras
05_computer_vision/         CNNs, object detection, segmentation
06_generative_models/       PixelCNN, VAEs, GANs
```
