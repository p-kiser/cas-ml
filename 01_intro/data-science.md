# Data Science

Levels of data analysis:

1. Descriptive analysis: *What happened?*
2. Diagnostic analysis: *Why did it happen?*
3. Predictive analysis: *What will happen?*
4. Prescriptive analysis: *How can we make it happen?*

Data Science / ML work is:

- 80% data
- 20% modeling and evaluation

Analysis vs ML:

- Descriptive analysis: no decisions
- Statistics: few decisions
- Machine Learning: many decisions

Roles:

- Data Analyst: queries
- Data Scientist: ...
- Data Engineer: data flow, transform
- ML Engineer: related to devops and software dev
- AI Engineer: orchestrations between LLMs, combine components to services

Tools:

Python, Jupyter Notebook, Looker, Tableau, PowerBI
